# Retro Gaming AI Chatbot

An AI chatbot focused on general retro gaming, built with vanilla JavaScript and prompt engineering. It covers classic consoles, arcade history, 8-bit and 16-bit games, cartridge culture, handhelds, and preservation — making it a broader gaming portfolio piece than a single-title bot.

- **Hosting:** Vercel / Netlify / GitHub Pages friendly

## Prompt engineering notes
- System prompt scopes the bot to 1970s–1990s retro gaming
- Fallback guidance nudges users back into scope instead of guessing
- Topic packs map keywords to precise historical answers for demo mode
- This makes interviews easy: you can explain system prompts, fallbacks, and scope control
